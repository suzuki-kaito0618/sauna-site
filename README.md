# 仙台のサウナ（施設紹介サイト）

宮城県仙台市のサウナ施設を紹介する個人サイトです。実際に訪れた施設の魅力、特徴、おすすめポイントをまとめています。

## 使い方（開き方）

### いちばん簡単（推奨）

`sauna-site/index.html` をブラウザで開くだけで表示できます。

### ローカルサーバで見る（任意）

相対パスの挙動を安定させたい場合は、簡易サーバで起動できます。

#### PowerShell（Pythonが入っている場合）

```powershell
cd .\sauna-site
python -m http.server 5173
```

その後、ブラウザで `http://localhost:5173` を開いてください。

## カスタマイズしたい場合

- `index.html`: 文章・セクション・リンク（メールアドレス等）
- `styles.css`: 色・余白・レイアウト
- `main.js`: スマホメニュー、コピー機能などの軽い動き
